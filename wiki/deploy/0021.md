## 5. 특이사항

- **코드 서명 스킵**: 유효한 "Developer ID Application" 인증서가 없어 macOS 코드 서명이 생략됨. 로컬 개발/테스트 용도로는 문제없으나, 배포 시에는 Apple Developer 인증서 설정이 필요함.
- **author 필드 누락**: `package.json`에 `author` 필드가 정의되어 있지 않아 electron-builder가 경고를 출력함. 기능상 문제는 없으나 정식 배포 시 설정 권장.
- **빌드 타겟**: macOS arm64 DMG (APFS 포맷, macOS 10.12+ 지원).
- 모든 타입 체크, 테스트, 빌드가 에러 없이 정상 완료됨.
